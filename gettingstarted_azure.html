MICROSOFT WINDOWS AZURE

Windows Azure services can be easily blended with mydigitalstructure.com.

  • Using Azure Websites you build a front end (view-controller) that you have full control over and then interface with the mydigitalstructure webapi (model).  First 10 Websites are free.

    eg create a marketing website and have enquires and opportunties automatically saved into mydigitastructure and then accessed via the ERP interface for further management.

  • Using Azure Storage Services to hold media and link to mydigitalstructure. more...
     
  • Using Azure Cloud Services to build out mydigitalstructure webapi with full control.
