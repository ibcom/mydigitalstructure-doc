NODEJS

A server side javascript engine.

A typical use of nodejs with mydigitalstructure,is as a transformation or automation agent (proxy).

The first step is to install nodejs or use a hosted environment like AWS.

We have created an open source nodejs module to assist with building apps that work with the mydigitalstructure service/api.
