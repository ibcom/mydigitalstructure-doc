GETTING STARTED
WEBAPP BUILD OPTIONS

You can engage with the mydigitalstructure webapi/platform at a number of levels depending on the user needs
and desired user experience.

If you are a systems integrator or reseller of IT systems, we highly recommend you check out alt-enter.com,

a ready to go all-of-enterprise app based on mydigitalstructure.com.

The amount of building work required in each of the options below generally gets greater as the number increases - in some cases it might be a combination of a few,  e.g. LOOK & FUNCTION.

The platform is highly flexible, so if you can imagine an app or have a user need, mydigitalstructure.com can help to make it real.

Don't forget to follow us at twitter.com/ibComMYDS or visit the community discussion forum if you have any questions and check out the Getting Started page for a stack of tricks & tips.

You can use the simple mydigitalstructure IDE or use your preferred IDE for the environment you are developing for.

We also recommend:

  • Uses & architecture diagram
  • Reading an overview of building using MVC techniques
  • Watching this video by Rob @ 1upIT about building your first app.
