GETTING STARTED
STRUCTURES

Structures generally define the information stored with in an object eg for a contact it includes firstname, surname, phone etc.

With-in the mydigitalstructure service you can use structures to extended an existing object. 

You can then store and search for information managed with-in these structures, in the context of an object.

Structures are made up of categories and elements (which can include options - eg if select list)





USING TO MANAGE OBJECT INFORMATION

A structure element can be updated like any other standard attribute via name pair value.

&seNNNN=[value]

where NNNN is the element reference number.


USING WITHIN AN ADVANCED SEARCH

You can add attributes to the select and filter list as per any other attribute.
