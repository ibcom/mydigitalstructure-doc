EXAMPLE
openAFSL

Radically simplifying the monitoring of advisers.

The advise given by AFSL registered advisers can have a huge impact on the individuals that the advise relates to.

openAFSL is a fully hosted end to end solution, managing:

  • Advisers, including skills
  • Auditors
  • Reviews
  • Issues
  • Correction Actions (Remediation)
  • Reporting

Access is available on tablets for in-field reviewing and desktops for in-office use.

The app is fully hosted, so no on-premise set up is required and the app is pre-populated with a 150 element review template.

The app is built on top of the world class ibCom mydigitalstructure platform, so integration and extensibility is inherent.

All in all, with a few set up steps you can have a full review and reporting system for your AFSL license based organisation.
