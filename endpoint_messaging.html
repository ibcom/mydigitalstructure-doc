  /rpc/messaging/  
  View and send emails, send SMSs and use conversations to manage blogs, wikis etc with post and comment structure.  
     
  Endpoint methods can be incorporated into any http compliant device eg Android, iPhone, for any place, any time collaboration - combine with websockets to build a "push" solution.
