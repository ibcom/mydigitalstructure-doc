  OPPORTUNITY_MANAGE OPPORTUNITY Endpoint
  /ondemand/opportunity/?method=OPPORTUNITY_MANAGE  
  PARAMETERS TYPE  
  Id Numeric  
  Remove Numeric 1/0
       
  AdvisorContactPerson Numeric As per CONTACT_PERSON_MANAGE
  AllowSourceToSee Text (1) Y/N [default]
  AllowSourceToSeeOrderActions Text (1) Y/N [default]
  AllowSourceToSeePolicyActions Text (1) Y/N [default]
  AllowSourceToSeeProjectActions Text (1) Y/N [default]
  Amount Numeric  
  Area Numeric  
  BusinessName Text (100)  
  Description Text (1000)  
  Email Text (100)  
  FirstName Text (50)  
  For Numeric 1=My Business [default],2=Other Business
  MailingAddress1 Text (100)  
  MailingAddress2 Text (100)  
  MailingCountry Text (50)  
  MailingPostCode Text (10)  
  MailingSuburb Text (50)  
  ManagerUser Numeric As per SETUP_USER_MANAGE
  Mobile Text (50)  
  OwnerContactBusiness Numeric As per CONTACT_BUSINESS_MANAGE
  Phone Text (50)  
  Probability Numeric Should be 0 - 100, but you can use other values if you insist. If between 0 and 1, it automatically gets converted to a percentage
  ProcessingDate Date  
  ProcessingNote Text (2000) Stored in the Processing History subtable
  ProcessingStatus Numeric You need to define your own. Currently, there is no method for this, contact ibCom for more information.  If you change this, a record gets written to the OPPORTUNITY_PROCESS_STATUS_HISTORY area, along with the passed Processing Note & Date
  Product Numeric As per PRODUCT_MANAGE
  Reference Text (50) Defaults to OPPnnnnnn
  RequestByContactBusiness Numeric As per CONTACT_BUSINESS_MANAGE
  RequestByContactPerson Numeric As per CONTACT_PERSON_MANAGE
  Source Numeric There are some defaults (eg 1=Word of Mouth), plus you need to define your own. Currently, there is no method for this, contact ibCom for more information
  SourceNote Text (100)  
  StartDate Date  
  Status Numeric 1=Open [default],2=Closed,3=Cancelled
  Surname Text (50)  
  Type Numeric You need to define your own. Currently, there is no method for this, contact ibCom for more information
  RETURN  
  id
