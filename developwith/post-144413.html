Automating Advanced Search Results
30 Mar 2012
mark.byers

Cassie has just completed some great work in regards to using the new generic functionality that allows the results of an advanced search to create a custom email that is personally sent to each person using the data returned in the results.

This gives incredible power to the designer/developer when creating work flow alerts and also user created bulk communications.

eg

1. User defines search
2. Search results returned
3. User uses document editor to insert the "tags" based on the other data returned
4. User then clicks Send
5. The mydigitalstructure.com messaging engine then sends to each person in the original search.

A developer getting started guide is at:

http://mydsondemand.com/gettingstarted_results_automation
