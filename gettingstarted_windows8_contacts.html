THE WINDOWS 8 CONTACTS STORY

A story about how the Services Oriented Architecture is changing everything in regards to information applications - in the same way that the internet changes everything else.

NEED:

Build a contact manager for upcoming Windows 8 based on mydigitalstructure.com.

SERVICES:

Freelancer.com & mydigitalstructure.com

ASSEMBLY RESOURCES:

  • Microsoft Windows 8 - user experience client
  • paul142 - an expert javascript programmer on Windows 8 from the Ukraine, rate: $US15/hour.
  • paint.net - to draw the interface
  • Microsoft Word - to write the design document.


THE BUILD:

  1. Screens drawn using paint.net - 1 hour
  2. Business logic and tips on using mydigitalstructure.com - 2 hours
  3. Submit project on freelancer.com and hire paul142 - 5 min
  4. Confirm with paul142 that good to go - 5 min
  5. Project coding - 12 hours  over 2 days.
  6. Payment made.


Total combined time for "production ready" Windows 8 prototype - 15 hours and 10 minutes.
