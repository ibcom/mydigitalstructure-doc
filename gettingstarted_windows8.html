BUILDING AN APP IN WIN8/RT,
BASED ON MYDIGITALSTRUCTURE

Once you have Windows 8 installed, then you can start to build your first Metro style app.

Given the mydigitalstructure commitment to the model-view-controller (MVC) framework, a lot of the existing Javascript/jQuery based code can be leveraged into the Windows 8/RT user experience using WinJS.

You can choose to use the Windows 8/RT native controller functions or reuse your existing jQuery based controller code and then use the Windows 8 IE10 based engine to render the view using js/HTML5.

You can build the app in c# or any other scripting/coding language - the steps below are focused on Javascript/HTML5.

STEP 1

Read Microsoft Windows Metro Style app development & Building your first Windows Metro style app with JavaScript.

STEP 2

Using Visual Studio 11 Express get the 1blankspace.com default scripts or just build directly on mydigitalstructure and get the jQuery scripts (must be local) and build a great user experience within Windows 8!

We'd love to see what you're building, so don't hestitate to contact us with what you've built.
