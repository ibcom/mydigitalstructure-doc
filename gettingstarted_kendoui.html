GETTING STARTED
KENDO UI

Kendo UI combines everything needed for modern JavaScript development, including rich UI Widgets, a powerful DataSource, an MVVM framework, universal Drag-and-Drop, Templates, and Themes.

Building on top of the popular jQuery JavaScript library, Kendo UI leverages the latest in CSS3, HTML5, and JavaScript web standards while still delivering broad support for all major browsers and mobile devices like iOS, Android and BlackBerry.

It allows you to build apps based on the mydigitalstructure.com model even faster!

  • Complete framework for HTML5 apps
  • Ready for touch devices
  • Native UI on every mobile device
  • Ready for app stores
  • Data visualisations powered by HTML5


Kendo UI, mydigitalstructure.com & Webapps

Given the MVC architecture that both platforms support and jQuery dependency - using the Kendo UI Widgets and DataViz is a very natural fit.

To get started:

  1. Sign up to the Kendo UI Complete trial today!
     
  2. Check out the demos and ThemeBuilder
     
  3. Confirm dependancies


Referencing the files:

  • You can attach the downloaded js & css files to your own site, ie /site/1234/

  • Use the Kendo UI CDN
     

KENDO UI LICENSING

Kendo UI by Telerik is a commercial framework licensed royalty free, per developer. To use Kendo UI with the mydigitalstructure platform/webapi, you must obtain the proper license directly from Telerik. mydigitalstructure.com is a platform only and thus does not directly provide licensing. If you choose to develop with Kendo UI, please visit KendoUI.com to obtain the proper licenses.
