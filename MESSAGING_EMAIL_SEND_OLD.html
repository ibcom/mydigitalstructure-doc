  MESSAGING_EMAIL_SEND MESSAGING Endpoint
  /ondemand/messaging/?method=MESSAGING_EMAIL_SEND  
 

Send an email.

Before sending an email the "fromemail" address must be verified.

  
  PARAMETERS TYPE NOTES
  subject Text  
  message Text Message to be sent.
  to Text ; delimted list of emails
  id Text , delimted list of contact person ids
  cc Text ; delimted list of emails to cc
  bcc Text ; delimted list of emails to bcc
  fromemail Text Optional. Defaults to the email address linked to your user record
  document Numeric ID of document to send.
  draft Numeric ID as returned from MESSAGING_EMAIL_DRAFT - used for sending attachments
  object Text System object to link saved email to
  objectcontext Text ID of object instance to link to
  bccfrom 1 If set to 1, send bcc to from person
  attachmentobject Numeric Get attachments from this system object
  attachmentobjectcontext Numeric ID of object instance to get attachments from
  attachmentlink Numeric ID of a specific attachment. Also need to pass attachmentobject & attachmentobjectcontext
  RETURN  
  status SENT
