  ADVANCED SEARCH - STANARD SYSTEM PARAMETERS Advanced Search
 
These parameters are available on all advanced searches.		  
  AVAILABLE FIELDS TYPE  
  CreatedDate Date Date the item was created
  CreatedUser Numeric ID of the user who created the item
  CreatedUserText Text Name of the user who created the item
  ETag Text md5 hash which can be used to determine if a record has been changes since you last retrieved
  ModifiedDate Date Date the item last modified
  ModifiedUser Numeric ID of the user who last modified the item
  ModifiedUserText Text Name of the user who last modified the item
