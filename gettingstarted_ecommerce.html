GETTING STARTED
ECOMMERCE & ONLINE SHOPS

You can create incredible ecommerce webapps - from simple product stores through to full order processing,
stock management and financial accounting - using mydigitialstructure.

You can build any interface style you want.

3 typical uses include:

  • Public store

  • Customer account management

  • Administration 


PUBLIC STORE

Display products - search engine compatible with deep links.

Once you have you creative design complete you need to register a site with in your space.


CUSTOMERS

Log on to secured area with order viewing, status checking, profile updating etc

If you want to give customers accessing to orders then you register another space and setup a login page.


ADMIN

Order processing, CRM, activity tracking, invoicing, financials etc

The simplest we to get this going is to use the 1blankspace example order management scripts.

EXAMPLE WEBSITES:

  • popet.com.au
  • emufire.com.au
  • ledlighting.com.au


TECHNICAL:

  • Create category and product template documents using dynamic tags.  Set document type = 'System Template'.
  • Category url: /mycategories/[category id]/[search]
  • Product urls: /myproducts/[productid]/[your own context - available in variable msOnDemandContextId]
  • You can create variants of products using SETUP_PRODUCT_VARIANT_MANAGE
  • You can retrieve variants using SITE_PRODUCT_VARIANT_SEARCH
  • Full method list for shop construction
