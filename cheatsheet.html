Endpoints https://au.mydigitalstructure.com for off-platform access.

If you're developing a webapp based on 1blankspace.com, then you can just use the relative path, as it is automatically in context of the 1blankspace.com domain which has access to the endpoints.

  
/ondemand or /rpc? The method details page shows the full URL which contains the sub-endpoint, by default it is /ondemand.  Since September 2012 we have been building up our /rpc sub-endpoint in preparation for our /rest endpoint.  So the latest versions of methods can be found at /rpc - the methods details page will be marked with an RPC icon when you need to use /rpc.

You can also reference /jscripts/1blankspace.rpc-2.0.0.json for a list of /rpc methods.
